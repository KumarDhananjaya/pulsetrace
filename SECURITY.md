# Security Policy

## Supported Versions

We provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | ✅ Yes             |
| < 1.0   | ❌ No              |

## Reporting a Vulnerability

If you discover a security vulnerability within PulseTrace, please do not open a public issue. Instead, report it to the maintainer:

- **Email**: security@pulsetrace.io (Dummy)
- **GitHub**: Report a vulnerability via GitHub's Security tab (if enabled).

We will acknowledge your report within 48 hours and provide a timeline for a fix. Thank you for helping keep PulseTrace secure!
